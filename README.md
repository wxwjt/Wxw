# 工具总和网站

一个精美的现代工具集合网站，包含多种实用工具和复古像素风格的贪吃蛇游戏。

## 项目特点

### 🌟 主要功能
- **工具集合展示**：展示多种实用工具，包括像素艺术生成器、代码格式化工具、科学计算器等
- **复古像素风贪吃蛇游戏**：完整实现的贪吃蛇游戏，具有像素风格和半调图案设计
- **响应式设计**：适配各种屏幕尺寸，支持桌面端和移动端
- **现代化UI**：采用玻璃态效果、渐变色彩和流畅动画

### 🎨 视觉设计
- **色彩方案**：蓝色到紫色的渐变主题，营造科技感
- **玻璃态效果**：半透明背景和模糊效果，提升视觉层次
- **动画效果**：平滑的过渡动画、悬停效果和滚动动画
- **像素艺术元素**：融入像素风格设计，特别是在贪吃蛇游戏中

### 🎮 贪吃蛇游戏特色
- **经典玩法**：使用方向键控制蛇的移动，吃到食物增长得分
- **视觉效果**：蛇身颜色随得分变化，食物带有闪烁效果
- **交互反馈**：吃到食物时有像素爆炸效果，增强游戏体验
- **移动支持**：支持触摸屏控制，适配移动设备

## 技术栈

- **HTML5**：语义化HTML结构
- **CSS3/Tailwind CSS**：现代化样式设计
- **JavaScript**：游戏逻辑和交互功能
- **Font Awesome**：图标库
- **GSAP/AOS**：动画效果

## 如何使用

1. 克隆或下载项目到本地
2. 打开 `index.html` 文件即可访问完整网站
3. 无需安装任何依赖，所有资源通过CDN加载
