# 工具总和网站

一个精美的工具总和网站，包含多个功能完整的实用工具和游戏。

## 功能特点

### 实用工具
1. **代码格式化工具** - 支持JSON、HTML、JavaScript、CSS代码格式化和压缩
2. **像素艺术生成器** - 创建和编辑像素艺术，支持多种画布大小和颜色工具
3. **科学计算器** - 功能强大的科学计算器，支持三角函数、对数、阶乘等
4. **颜色选择器** - 选择和转换颜色，支持HEX、RGB、HSL格式，生成调色板
5. **单位转换器** - 转换长度、重量、温度、面积、体积、速度、数据存储和时间单位
6. **密码生成器** - 生成安全密码，评估密码强度，自定义密码选项
7. **二维码生成器** - 生成自定义二维码，支持颜色和大小调整
8. **数据可视化工具** - 创建和显示图表，支持多种图表类型

### 游戏
1. **贪吃蛇游戏** - 经典贪吃蛇游戏，支持多种模式和难度选择
   - 经典模式
   - 障碍模式
   - 速度模式
   - 多食物模式

### 其他功能
- **响应式设计** - 适配桌面端和移动端
- **深色/浅色主题** - 支持主题切换
- **现代化UI** - 玻璃态效果、渐变色彩、流畅动画
- **本地数据处理** - 所有数据在浏览器本地处理，保护用户隐私
- **免费使用** - 所有工具完全免费，无需安装

## 技术栈

- **HTML5** - 页面结构
- **CSS3/Tailwind CSS** - 样式和布局
- **JavaScript** - 交互功能
- **Font Awesome** - 图标库
- **Chart.js** - 数据可视化
- **GSAP** - 高级动画
- **AOS** - 滚动动画
